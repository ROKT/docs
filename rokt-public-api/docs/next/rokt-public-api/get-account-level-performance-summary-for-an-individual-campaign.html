GET Account Level Performance Summary for an individual Campaign

Path

/reporting/accounts/{accountId}/campaigns/{campaignId}/summary

Description

Returns performance metrics for a campaign for a given time period, time zone and currency.

When to call

Call this API endpoint to receive campaign level performance metrics, like impressions, referrals, conversions for a specified time period, timezone and currency.

Request

dateStartstringStart time of the requested date/time rangetruedateStart=2020-03-01T00:00:00.000

Response

200 OK
