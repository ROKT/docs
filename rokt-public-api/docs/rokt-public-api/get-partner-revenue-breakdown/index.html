GET Partner Revenue Breakdown

Path

/reporting/accounts/{accountId}/transactions/breakdowns/revenue

Description

Returns the revenue of a partner over a period of time broken down into device, gender and age bracket segments.

When to call

Call this API endpoint to get the revenue breakdown of a partner based on criteria like age, gender and device.

Request

KeyTypeDescriptionRequired?Example
dateStartstringStart time of the requested date/time rangetruedateStart=2020-03-01T00:00:00.000
dateEndstringEnd time of the requested date/time rangetruedateEnd=2020-03-31T23:59:59.000
currencystringCurrency code in which you will receive monetary metrics.truecurrency=USD

Response

200 OK
