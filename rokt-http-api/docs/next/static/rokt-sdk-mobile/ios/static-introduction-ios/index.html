Introduction

The Rokt SDK for iOS enables you to integrate Rokt into your native iOS mobile apps in order to drive more value from, and for, your customers. The SDK is built to be lightweight, secure, and simple to integrate and maintain, making it minimal lift for your engineering team.

For Rokt Partners, the SDK empowers you to optimize a front-end user experience through commerce optimizations and third party offers. When instructed by your app, the Rokt SDK for iOS is able to display Rokt Placements either as overlays, on top of your view, or as embedded features of the existing view. Rokt Placements are customizable via the Rokt platform, providing a selection styling options (e.g. font, buttons etc.). In addition to this your app is able to dictate the consumer attributes that are passed to Rokt. These data attributes are used to personalise the Rokt experience and optimise offer selection for each individual user.

For Rokt Brands, the SDK closes the loop on conversion tracking in-app, enabling Rokt to better optimize campaigns towards users that are more likely to convert. When instructed by your app, the Rokt SDK for iOS will record a conversion event. Based on the consumer attributes provided to Rokt, the Rokt system will attribute this conversion to the appropriate referral event allowing improved targeting and automated bidding in future.
